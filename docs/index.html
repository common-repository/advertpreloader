About the plugin

AdvertLoader is a plugin designed to increase your earnings by using adverts which will be shown to users just before all other content loads. The plugin can also be used to notify users about some important information before showing the page content.

How does it work?

The plugin is fully configurable and can be set to appear on posts, pages or any other type of entry for which you activate the preloader. Before users can see the content of a page (texts, pictures, video) the plugin displays a message stating that the content is currently loading, providing you with an option to put an advert, a picture, or a piece of text just below the message. You can define almost all parameters regarding the look of the preloader (from the loading animation, the duration of the loading to the various ways the preloader can be shown).

The demo can be seen here: http://ivanmaricic.from.hr/moje-misli-moji-mah-glupost/ In the demo example, the preloader is set to display a button which will reveal the content (upon clicking on it), but the button itself will only appear ten (you can change that!) seconds after visitors open the page.

  • When loading a video. You can inform users that the video is being loaded, while showing them an advert or an important message at the same time.
  • To show the terms of service or protect users from inappropriate content. If your page features adult content you can set the preloader to display a ten second message which will inform potential visitors that the site is intended for a 18+ audience, asking them to click on the continue button in order to view the content.

Usage guide

1. Installation

Installation is very simple, just unpack downloaded archive file, and upload plugin directory into wp-content/plugins. After plugin is uploaded login into wp-admin, and activate plugin from Plugins -> Installed plugins (http://yoursite.com/wp-admin/plugins.php)

2. Wordpress configuration and options for plugin

After successful installation and activation of the plugin, the icon for configuration and customizations will appear on the bottom left side of the menu in the WP dashboard. Click on AdvertLoader Config to open configuration page. When you open the settings you will see options for plugin like on bottom image. The plugin is very easy to use but we will pass one by one field to make sure you understand the options which plugin contains:

1. In the left field enter time in seconds during which the visitor sees the content loaded in the preloader. On the right menu, choose whether to view the content automatically after the expiry of the time or visitor have to click on a button to reveal the content.

2. Choose where to show preloader.
If you choose the first option (recommended), preloader will appear only on the content page (like a single page), but if you choose the second option, preloader will appear on all other pages that contain a summary of the contents (eg. category page, archive, homepage etc.).

3. Custom text that is displayed while running out of time that we set ourselves on first option.

4. Custom text that is shown after expiration time of preloader.

5. Custom text in the button which display the content (only if selected option to display button for discovering content).

6. Preloader content (Your ads or banner, image, customized HTML code of your choice, etc.).

7. Optionally select an progress animation that best fits your website. Feel free to experiment!

8. Select whether the preloader appear only on posts or just the pages or on all posts and pages.
If you choose the last option "Only on selected posts and pages" you have to check "Show Preloader" checkbox under "Edit Post" form. The new meta-box will appear in Editor where you choose "Yes" to show preloader or "No" to hide preloader for certain posts. If you don't see the meta-box in the "Edit Post" page, click on the button "Screen Options" that is located on the top-right of Wordpress administration and check "AdvertPreloader options" to be shown on screen.

9. "Save Changes" and see the results.

That's it. Your preloader is now ready and active! Enjoy!
